# SailorM Dungeon Hunters

Chrome/Edge расширение для автоматизации события ReManga **Охотники подземелий**.

Расширение не содержит привязки к конкретному аккаунту. Оно работает с текущей авторизованной сессией ReManga в браузере и хранит только локальные настройки в `chrome.storage`.

## Возможности

- Показывает статистику события: ранг охотника, XP, MP, монеты, успешные и провальные прохождения.
- Запускает подземелья по выбранному режиму.
- Забирает готовые награды.
- Завершает доступные мини-игры события.
- Собирает доступные награды адвент-календаря.
- Учитывает локальную оценку восстановления MP.
- Поддерживает автопилот через `chrome.alarms`.
- Позволяет включать или отключать зелья для отдельных режимов.
- Добавляет movable GUI прямо на страницу `dungeon-hunters`.

## Режимы прохождения

- `safe` - безопасно, свой доступный ранг.
- `xp` - выбор по ожидаемому XP с учетом шанса успеха.
- `highest` - самый высокий доступный ранг.
- `fast` - быстрый фарм доступного подземелья.

## Настройки

Во вкладке `Settings` доступны:

- `Зелья` - разрешает использовать зелья при запуске подземелья.
- `Сбор` - автоматически забирает готовые награды.
- `Мини-игры` - автоматически закрывает мини-игры успешным результатом.
- `Календарь` - собирает доступные награды адвент-календаря.
- Минимальный ранг, с которого разрешено использовать зелья.
- Отдельные переключатели зелий для режимов `safe`, `xp`, `highest`, `fast`.

## Установка

1. Скачайте или клонируйте репозиторий.
2. Откройте `chrome://extensions/` или `edge://extensions/`.
3. Включите режим разработчика.
4. Нажмите `Load unpacked` / `Загрузить распакованное расширение`.
5. Выберите папку с расширением.
6. Откройте `https://remanga.org/dungeon-hunters`.

## Приватность

В репозиторий не включаются:

- ID конкретных пользователей.
- Токены, cookies или данные авторизации.
- Локальные профили браузера.
- История выполнения и настройки из `chrome.storage`.

Расширение использует авторизованную сессию текущего браузера и отправляет запросы только к `remanga.org` и `api.remanga.org`.

## Предупреждение

Автоматизация действий на сайте может нарушать правила сервиса. Используйте расширение на свой риск.
