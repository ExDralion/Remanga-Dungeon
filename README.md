# SailorM Dungeon Hunters

Расширение для события ReManga `Охотники подземелий`.

## Что умеет

- Показывает профиль охотника, MP, XP, монеты и статистику runs.
- Запускает один цикл автоматически:
  - если активна мини-игра, закрывает ее успешным результатом;
  - если обычный данж готов, забирает награду;
  - если активного данжа нет, выбирает новый по выбранному режиму.
- Поддерживает автоцикл через `chrome.alarms` раз в минуту.

## Режимы

- `safe` - свой доступный ранг.
- `xp` - лучший ожидаемый XP с учетом шанса успеха.
- `highest` - самый высокий доступный ранг.
